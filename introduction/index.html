About this tutorial

This guide is intended to teach people who have no previous programming experience how to build web-based applications.

What is a web applicaion?

Good question! A web site is a collection of web pages that link together. Web sites are purely informational. No programming is required to build web sites, only knowledge of HTML and possibly CSS. A web application allows the user to interact with the web site, by entering information into a form, displaying information from a database, etc. What you are reading right now is just a web site. GMail, Facebook and Amazon are examples of web applications. Ruby on Rails is a framework that was built to help you build web applications.

It's more than just Ruby on Rails

Without any prior programming experience, learning on Ruby on Rails can be very challenging. One of the biggest reasons why is that you need to know many other things besides just Ruby on Rails. This guide starts you out with the basics to give you a good foundation to build on.

What you need

By far, the most common platform developers building Ruby on Rails use is Mac OS X1. These instructions will only cover how to use a Mac. If you only have a Windows or Linux PC to use for development, you may be able to follow some other tutorials online about how to get things installed and up and running and then follow the rest of this guide, but I recommend getting a Mac.

The instructions for this guide also cover the latest verison of Mac OS X, which is 10.9 Mavericks. Mavericks is a free upgrade, available in the Mac App Store, so make sure you have installed that before proceeding.
